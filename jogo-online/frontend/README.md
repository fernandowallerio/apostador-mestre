# 🚀 Batalha Espacial - Protótipo

Um jogo de estratégia espacial por turnos desenvolvido em JavaScript. O projeto foca em mecânicas de planejamento simultâneo e combate tático.

## 📂 Estrutura do Projeto

O projeto está organizado para facilitar a manutenção e escalabilidade:

- **/frontend**: Contém os arquivos de interface e estilo.
  - `index.html`: Estrutura principal da página.
  - `style.css`: Estilização visual e cores do jogo.
  - `ui.js`: Gerencia as interações do usuário e desenhos no canvas.
- **/game**: Contém a lógica central (o "cérebro").
  - `engine.js`: Processa as regras, movimentos e sistemas de combate.

## 🛠️ Tecnologias Utilizadas

- **HTML5 Canvas**: Para renderização gráfica.
- **JavaScript (ES6+)**: Lógica e motor do jogo.
- **CSS3**: Layout responsivo e visual moderno.

## 🎮 Como Jogar

1. O Jogador 1 planeja os movimentos e alvos das suas naves.
2. O Jogador 2 realiza o mesmo procedimento.
3. Clique em **Executar Turno** para ver as ações acontecerem simultaneamente!
